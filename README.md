# 世紀末異郷人 —— SEIKIMATSU · IHOUJIN

> 1999 · 夏 · 石を推す者たち  
> The Stranger at the End of the Century

---

## About

一部関于**世紀末**、**哲學**與**選擇**的視覺敘事體驗。  
純 HTML / CSS / JavaScript 實現，無依賴，可直接部署於 GitHub Pages。

基於原創設定集《世紀末異郷人》。

## Features

- ✨ **CRT 啟動序列** — 復古電視機開機質感
- 🎬 **4 段式 OP 動畫** — 1999 → 哲学症 → 人物剪影 → 題字
- 📚 **登場人物名鑑** — 六位論證者的完整檔案
- 📖 **哲学症・用語集** — 世界觀設定百科
- ⚔️ **命題展開戰鬥** — 引用・命題展開・辯證反轉・死亡重置
- 🔊 **Web Audio 生成環境音** — 蝉鳴 + 低頻 drone
